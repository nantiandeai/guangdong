广东省第十三届美术书法摄影作品联展（摄影金、银、铜奖）

时间：2015-09-21 来源：广东省文化馆 作者：广东省文化馆

活动介绍 展览展示

晚霞蒙蒙，远山如黛，叠翠葱茏，一道光照进山间田头，近处是青青禾田，远处是如画山川，山脚的村庄，小楼幢幢，富足安宁，描绘的俨然是盛世锦绣，岁月静好的图景。这里是韶关仁化 的丹霞山下，广东一处普通的乡村。

这是一幅名为《天上人间》的摄影作品，由摄影爱好者张标兵拍摄，是“美丽广东、活力广东”摄影大赛中的特等奖作品。获奖理由并不单纯是因为画面拍的极美，更重要的是，其中透露出 的人与自然相得益彰，保护自然，在美丽环境中自得安乐生活的理想，暗合了本次摄影大赛“开拓创新、诚信务实、绿化广东，建设美丽家园”的主旨。

“美丽广东、活力广东”摄影大赛由广东省人民政府新闻办公室主办，是为对外介绍我省加快转型升级、建设幸福广东成果而举办的摄影界盛事，至今已经是第四届。本届大赛从今年3月份启 动面向社会征稿至7月31日截止，共征集到摄影作品2万余幅，吸引了众多摄影界专业人士和社会民众积极参与。这次大赛共分5个组别，分别为“多彩大地、岭南文化、经济建设、和谐社会、美 丽人物”，经专家认真评选，按5个组别各评出一等奖1名、二等奖2名、三等奖3名、优秀奖24名，以及特等奖1名。151幅获奖作品虽不能道尽广东之美，却能让观者感同身受，窥见一斑。广州 珠江新城CBD的建筑之美，岭南“四水归堂、青云巷深”的文化之魅，广东高铁建设的飞腾之美，外来工在广东安居乐业的生活之美，广东各种平民英雄的善举之魅都尽在其中。目前，“美丽广 东、活力广东”摄影大赛151幅获奖作品已于南方网、金羊网、21cn、腾讯•大粤网、南方英文网等网站设专题展示，欢迎读者前往浏览欣赏。
